# Cozy Journeys

Cozy Journeys is a small travel journal made for saving the places, moods, photos, and tiny details that are easy to forget after a trip ends.

I built it because I wanted something warmer and  meant to feel more like a soft Pinterest scrapbook sitting on a desk.


## How it works

The page is split into two simple sections.

On the left, there is a notebook  form where you can add:

- A destination
- The date of the trip
- Your mood
- Photo url ( add from cdn.hackclub.com )
- The memory



Once you press Pin to Journal , the memory appears on the right as a Polaroid card. Each card shows the photo, destination, date, and mood with the entire story.


Entries can also be removed using the small delete button on each card.

## Built with

- HTML
- CSS
- JavaScript


## LocalStorage

Your journal entries are saved directly inside your browser.

This means they stay there after refreshing or closing the page, as long as the browser data is not cleared.



## Using photo links

use cdn.hackclub.com or #cdn from slack
